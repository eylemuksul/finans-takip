# 💰 Kişisel Finans Takip Sistemi

Python, Flask ve SQLite kullanılarak geliştirilmiş; kullanıcıların gelir ve giderlerini anlık olarak kaydedip yönetebileceği, bütçe takibi yapabileceği web tabanlı bir finansal yönetim uygulamasıdır.

🔗 **Canlı Uygulama Linki:** [https://finans-takip-0yc5.onrender.com](https://finans-takip-0yc5.onrender.com)

---

## 🚀 Özellikler

* **Kullanıcı Kayıt ve Giriş Sistemi:** Güvenli şifreleme altyapısı ile her kullanıcının kendi verilerini güvenle saklaması sağlanır.
* **Gelir/Gider Yönetimi:** Harcamaların miktar, kategori, tarih ve açıklama eklenerek kolayca kaydedilmesi ve listelenmesi.
* **Dinamik Dashboard:** Toplam gelir, toplam gider ve mevcut net bakiye durumunun anlık olarak özet kartları üzerinden takibi.
* **İşlem Düzenleme/Silme:** Geçmişe dönük finansal hareketlerin kolayca güncellenebilmesi.

---

## 🛠️ Kullanılan Teknolojiler

* **Backend (Arka Plan):** Python, Flask
* **Database (Veri Tabanı):** SQLite
* **Frontend (Arayüz):** HTML5, CSS3, Bootstrap
* **Deployment (Yayınlama):** Render

---

## 💻 Yerel Kurulum ve Çalıştırma

Projeyi kendi bilgisayarınızda çalıştırmak için aşağıdaki adımları takip edebilirsiniz:

 ### 1. **Projeyi Klonlayın:**
 
   git clone [https://github.com/eylemuksul/finans-takip.git](https://github.com/eylemuksul/finans-takip.git)
   cd finans-takip

### 2.Gerekli Kütüphaneleri Yükleyin:

pip install -r gereksinimler.txt

### 3.Uygulamayı Başlatın:

python uygulama.py
Tarayıcınızdan http://127.0.0.1:5000 adresine giderek uygulamaya erişebilirsiniz.
